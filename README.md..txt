# Olist E-Commerce Dashboard

## Cara Menjalankan Dashboard
1. Install semua library: `pip install -r requirements.txt`
2. Jalankan perintah: `streamlit run dashboard.py`
3. Dashboard akan terbuka di browser.

## Struktur Folder
- `dashboard.py`: Aplikasi utama
- `dashboard/`: Berisi data hasil cleaning
- `data/`: Berisi dataset asli
